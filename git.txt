# Para salvar todas suas alterações no github siga os passos abaixo:

1-Abra o terminal
2-Rode o comando git add .     (Precisa ser na pasta raiz, no caso curso_python_estudonauta)
3-Rode o comando git commit -m "mensagem do commit"
4-Rode o comando git push origin main  (comando que vai enviar todas as alterações para o seu githubgit)
